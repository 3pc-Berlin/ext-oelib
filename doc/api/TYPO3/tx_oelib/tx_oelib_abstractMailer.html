Abstract class 'tx_oelib_abstractMailer' for the 'oelib' extension.

Abstract class 'tx_oelib_abstractMailer' for the 'oelib' extension. This class declares the function sendEmail() for its inheritants. So they need to implement the concrete behavior. Regarding the Strategy pattern, sendEmail() represents the abstract strategy.
