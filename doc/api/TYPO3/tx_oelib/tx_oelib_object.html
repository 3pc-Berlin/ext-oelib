Class tx_oelib_object

Description
Description | Descendents | Methods (details)

Class 'tx_oelib_object' for the 'oelib' extension.

Class 'tx_oelib_object' for the 'oelib' extension. This class represents an object that allows getting and setting its data, but only via protected methods so that encapsulation is retained.
