Class 'tx_oelib_fakeSession' for the 'oelib' extension.

Class 'tx_oelib_fakeSession' for the 'oelib' extension. This class represents a fake session that doesn't use any real sessions, thus not sending any HTTP headers. It is intended for testing purposes.

  • author: Oliver Klee <typo3-coding@oliverklee.de>
